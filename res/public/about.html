📢 NekoBox by qr243vbi

We’re introducing NekoBox, a powerful and flexible open-source proxy client developed by qr243vbi.


🔧 What is NekoBox?

NekoBox is a lightweight, open-source desktop proxy client for Windows and Linux that lets you route and manage network traffic with fine-grained controls. It supports a range of modern proxy protocols and subscription formats, provides easy import/export of server lists, and offers both system-wide (TUN) and per-application routing using rule-based policies. Designed for power users, NekoBox emphasizes flexibility, low resource usage, and offline configurability while providing a simple and intuitive user interface.


🚀 Key Features:

- Support for multiple modern proxy protocols

- Easy import of server configs and subscriptions

- Full system proxy (TUN mode)

- Rule-based traffic routing

- Lightweight and open-source
